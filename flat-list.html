How to assemble a bike

  1. Press bottom bracket into frame
  2. Install cranks

  3. Install the drive train

    1. Attach casset to rear wheel
    2. Fit on front sprocket
    3. Install deralure
    4. Put on chain

Finish The Bike

  1. Install stem onto fork
  2. Install handlebars onto stem
  3. Install front and rear wheel
  4. Install seat and seat-post
